# wap-vty-notes

## 项目简介

`wap-vty-notes` 是一个用于归档与发布独立 HTML 页面的仓库。  
这些页面通常为笔记、参考或演示用途，不针对任何特定域名或网站。  
每个页面均为独立的静态 HTML 文件，可直接在浏览器中打开或通过 GitHub Pages 访问。

## 目录说明

```
wap-vty-notes/
├── index.html          # 根目录入口文件（如有）
├── pages/              # 主要页面存放目录
│   ├── example-1.html
│   ├── example-2.html
│   └── ...
├── assets/             # 公共资源（CSS/JS/图片等）
│   ├── styles/
│   ├── scripts/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：存放所有独立 HTML 页面，命名清晰易识别。
- `assets/`：页面引用的公共样式、脚本或图片资源，按需组织。

## 页面归档说明

- 每个 HTML 页面为独立单元，不依赖其他页面。
- 页面内容可能包含笔记、示例或临时归档信息。
- 部分页面可能包含外部资源（如 CDN 库），请确保网络可访问。
- 页面文件名及内容结构建议保持简洁，便于长期维护。

## 维护说明

- 新增页面：在 `pages/` 下创建新的 HTML 文件，并在必要时更新 `assets/` 中的公共资源。
- 修改页面：直接编辑对应 HTML 文件，注意保持与其他页面的兼容性。
- 删除页面：移除对应文件，并检查是否有其他页面引用。
- 建议每次提交附带清晰的 commit 信息，便于追溯变更。

## 使用方式

1. 克隆仓库到本地：
   ```bash
   git clone https://github.com/your-username/wap-vty-notes.git
   ```
2. 直接在浏览器中打开 `pages/` 下的 HTML 文件。
3. 或启用 GitHub Pages 功能，访问对应 URL 浏览。

## 许可

本项目仅用于个人归档与学习用途，不涉及任何商业或营销目的。  
如有引用外部资源，请遵循其各自许可协议。
